# Web Portfolio

This application develop use ReactJS, TailwindCSS, and Deploy use Netlify

### Step by step

* Install NodeJS

* Create Directory

```bash
mkdir my-web-project
```

* Install ReactJS

```bash
npm create-react-app .
```

* Install TailwindCSS

```bash
npm i -D tailwindcss
```

* Install RemixIcon

```bash
npm i remixicon
```

* Run App

```bash
npm start
```

* Build App

```bash
npm run build
```

* Live Website : [https://khalif-rizaldi.netlify.app/](https://khalif-rizaldi.netlify.app/)
